# Recruit Crawler Back

Koa로 구현된 Node.js 백엔드 어플리케이션입니다.
채용공고 크롤러 v2를 위해 제작되었습니다.

## 🖇️ 준비 및 확인사항

### 환경
- NODE: v18

## ⌨️ 명령어

### Install

```bash
  npm ci
```

### build

```bash
  npm run build
```

### Start

```bash
  npm start
```


## 🏷️ 환경변수

- `NODE_ENV`: production(default) 
- `PORT`: 3005
